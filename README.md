# Broom
> Keep your desktop clutter-free.

## Description

This script is a simple Desktop organizer that moves all files and folders from /Desktop to /Downloads/Rug.

## Installation

- Windows

```sh
> pip install broom
```

- Linux

```sh
> sudo pip3 install broom
```

## Usage

  ```sh
  > broom
  ```
  Admire your wallpaper once again.
